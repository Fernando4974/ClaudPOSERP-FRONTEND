Manual de Operación POS

Teclas Naranjas (Gestión)

  • RECEIPT: Alterna la emisión del ticket.
  • POST: Para registros posteriores a la venta.
  • PRICE / PLU: Ingreso manual de precios o códigos de producto.
  • OPEN: Apertura de cajón sin venta.

Teclas Rojas (Corrección)

  • C (Clear): Borra el número actual o detiene una alerta.
  • ERR CORR / CANCEL: Elimina el último artículo o cancela la transacción completa.
  • - : Aplica una reducción de monto fijo.

Teclas Azules (Transacción)

  • RA / PO: Recibo a cuenta (Entrada de dinero) / Pago (Salida de dinero).
  • CA/AMT: Finaliza la venta (Efectivo) y calcula el cambio.
  • T/S 1 - 2: Impuestos/Tasas específicas.
  • CR / CHK: Pagos con Crédito o Cheque.

Teclas Verdes (Ayuda)

  • HELP: Abre este menú de ayuda.
  • %- : Aplica un porcentaje de descuento.

Teclas Grises/Negras

  • TAX: Programación y consulta de impuestos.
  • SUBTOTAL: Muestra el monto acumulado en la pantalla azul.
  • ↑ / ↓: Navegación por la lista de productos en pantalla.
  • NS: "No Sale", abre cajón para cambios menores.
  • CLK#: Cambio de operador/cajero.
